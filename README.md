# Python Web Scraper

## Overview

A simple Python web scraper built using Requests and BeautifulSoup. This project extracts webpage titles and hyperlinks from any publicly accessible website.

## Features

- Extract webpage title
- Extract hyperlinks
- Display first 10 links
- Simple command-line interface
- Error handling for invalid URLs

## Technologies Used

- Python
- Requests
- BeautifulSoup4
